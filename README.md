<b>ExcellentCrates</b> is simple yet advanced Spigot crates plugin with a lot of unique and useful features with in-game
editor.

# Fork Description

This fork of ExcellentCrates adds [Lifesteal](https://www.spigotmc.org/resources/1-8-8-1-21-1-lifestealcore-the-premium-lifesteal-experience-now-with-custom-textures.101284/) functionality. Specifically, it adds a new crate that uses player's lifesteal hearts instead of crate keys to open the crate.
